Web quiz game where players compete live against each other, race the clock, and earn speed bonuses for fast answers.

  Features
  - 4 built-in quizzes across General, Technology, Geography, and Food categories (~22 questions each, randomized every game)
  - Speed bonus scoring — answer faster, score more
  - Live active players panel showing who's playing, their progress, and score in real time
  - Leaderboard saved after every completed game
  - Full Arabic translation with RTL layout and Zain font
  - Create your own quiz via a built-in form
  - Dark UI with Tailwind CSS,mobile responsive

  Tech stack
  - Laravel 13 + SQLite
  - Livewire v4 (reactive components)
  - Alpine.js (client-side timer, custom dropdown)
  - Tailwind CSS v4
  - Vite for asset bundling
